Pre-requirement:

1. Java Runtime Environment 5.0 or later.
	Your system should have Java Runtime Environment installed version 5.0 or later. If not, please go to http://java.sun.com/ and install it.
	
2. ant
	Your system should have ant installed. If not, please go to http://ant.apache.org/ and install it.

--

Build ZTermApplet:
 
	ant

--

Run ZTermApplet with appletviewer:

	ant ZTerm
